Scout

Your AI security team in a CLI. Find vulnerabilities before hackers do — free, local, no signup.

$ pip install scout-security
Python 3.10+ MIT License Zero Config AI Optional

The Problem

AI coding assistants write insecure code. Hardcoded secrets, SQL injection, missing auth — they ship it all. You can't afford a security team. But you can't afford to get hacked either.

Scout catches what AI introduces. Static analysis finds 80% of real issues instantly — no API keys needed. Optional AI confirmation reduces false positives further.

Quick Start

Install

pip install scout-security

Scan your project

scout scan ./my-project

Read the report

# Opens security-report.md with:
# - Every vulnerability explained in plain English
# - Severity ratings (CRITICAL → LOW)
# - Exact fix instructions
# - Phased remediation plan (fix safe things first)

Features

🔍

Static Analysis

AST + regex patterns detect secrets, injections, XSS, missing headers, and vulnerable deps. Works offline, instantly.

🤖

AI Confirmation (Optional)

Send flagged snippets to Claude, GPT, or local Ollama. Reduces false positives. Never sends full files.

📋

Phased Fix Plan

Phase 1: zero-risk (secrets, deps). Phase 5: architecture. Fix safely, ship confidently.

🔒

Privacy First

Everything runs locally. No telemetry. No cloud. Your code never leaves your machine unless you opt into AI.

⚡

Fast

Scans 10,000 lines in under 10 seconds. No build step, no config file needed.

🧩

Extensible

Write a new scanner in 30 lines of Python. Inherit BaseScanner, implement scan_file(), done.

Built-in Scanners

CLI Commands

scout scan

scout scan <path> [OPTIONS]

Options:
  --model, -m    AI provider: anthropic | openai | ollama | none (default: none)
  --no-ai        Skip AI pass entirely (static scan only)
  --output, -o   Output path for the report
  --ollama-model Ollama model name (default: llama3)

scout fix (coming soon)

scout fix --phase 1 <path>

Applies zero-risk fixes: moves secrets to .env, bumps vulnerable deps.

scout validate (coming soon)

scout validate <path>

Re-scans changed files + runs tests to confirm fixes worked.

AI Providers

AI is always optional. Static analysis works perfectly alone.

ProviderSetupCost
None (default) Nothing — works out of the box Free
Ollama (local) ollama pull llama3 then scout scan --model ollama Free
Anthropic Set ANTHROPIC_API_KEY in .env ~$0.01/scan
OpenAI Set OPENAI_API_KEY in .env ~$0.01/scan

Add Your Own Scanner

from scout.scanners import register_scanner
from scout.scanners.base import BaseScanner
from scout.models import Finding
from pathlib import Path

@register_scanner
class MyScanner(BaseScanner):
    name = "my-scanner"
    description = "Detects my custom pattern"

    def scan_file(self, file_path: Path, content: str) -> list[Finding]:
        findings = []
        # Your detection logic here
        return findings

Then add one import line in scout/scanners/__init__.py. That's it.
